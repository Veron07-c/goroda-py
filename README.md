# Игра "Города" (Консольная версия)

Учебный проект по дисциплине "введение в программирование для лингвистов" (1 курс). Игра написана на Python с разделением на логические модули и интеграцией сторонних библиотек.

## 👥 Команда (Архитектура)
* **Лемешевская Вероника** — главный архитектор, разработка ядра игры (`main.py`), интеграция модулей, обработка логических тупиков, сосздание служебной функции расчета букв (`utils.py`).
* **Вахминова Татьяна** — парсинг городов с Википедии и работа с базой данных (`data_manager.py`).
* **Игнатенко Вероника** — валидация ходов игрока и обработка правил (`validator.py`).
* **Кублякова Елизавета** — разработка искусственного интеллекта, логики компьютера (`computer_ai.py`).

## 🛠️ Технологии и требования
* Python 3.10+
* Библиотеки: `requests`, `beautifulsoup4`

## 🚀 Инструкция по запуску
1. Склонируйте репозиторий.
2. Установите зависимости:
   ```bash
   pip install requests beautifulsoup4
3. Запустите главный файл:
   ```bash
   python main.py
